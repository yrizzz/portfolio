🚀 QUICK START
================================================================================

1. Choose Template:
   cd examples/api-templates
   ls -la

2. Copy & Customize:
   cp 01-basic-get.js my-api.js
   # Edit my-api.js

3. Or Generate:
   cd examples
   node generate-template.js

4. Test Locally:
   node -e "const api = require('./my-api.js').default; ..."

5. Upload:
   - Login as admin
   - API Management → Submit New API
   - Paste code or upload file

6. Test Live:
   curl http://yourapi.com/api/execute/v1/category/name
